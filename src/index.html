Inducción al parcial

En este parcial, pondremos a prueba los conocimientos sobre HTML5, evaluando temas clave como la estructura básica del HTML5, el uso correcto de atributos, la aplicación de etiquetas de agrupación y semánticas, y la correcta inclusión de enlaces. También deberán ser conscientes de la accesibilidad y optimización de sus páginas web.

Es fundamental que todas las decisiones, etiquetas y organización del contenido se documenten adecuadamente en el archivo README.md del repositorio de GitHub.

Instrucciones del Segundo Punto

  1. Crear el repositorio: Uno de los integrantes debe iniciar sesión en GitHub y crear un nuevo repositorio:
    • Nombre del repositorio: parcial-html-css
    • Descripción: Añadir una breve descripción del proyecto.
    • Ambos integrantes deben tener acceso.
    • Incluir el archivo README.md marcando la casilla correspondiente.
  2. Agregar a los colaboradores: Invitar al compañero como colaborador desde la pestaña Settings del repositorio.
  3. Clonar el repositorio: Clonar el repositorio en sus máquinas locales: 
    git clone https://github.com/usuario/parcial-html-css.git
  4. Estructura semántica: Crear una estructura HTML semántica respetando etiquetas como <header>, <nav>, <section>, etc.
  5. Reglas para los commits: Hacer al menos tres commits con mensajes descriptivos:
    • (feat): agregado el menú de navegación
    • (fix): corregidos los estilos de los botones
    • (docs): actualización del README.md
  6. Documentación en README.md: Documentar las decisiones importantes del proyecto.
